# homebrew-tap

Homebrew tap for [emprise](https://emprise.dev) — the AI partner for your terminal.

```sh
brew install senzalldev/tap/emprise          # CLI
brew install --cask senzalldev/tap/emprise-desktop  # Desktop app (macOS)
```

Upgrades: `brew upgrade emprise`. The formula tracks the latest
[release](https://github.com/senzalldev/emprise-app/releases) and is
updated automatically by the emprise release pipeline.
